# Fake Job Post Detection using Machine Learning

This project detects fraudulent job postings using machine learning and natural language processing.

## Techniques Used
- TF-IDF Vectorization
- Logistic Regression (baseline)
- Random Forest Classifier
- SMOTE for class imbalance handling

## Results
Accuracy: ~98%

SMOTE improved fake job recall significantly.

## Tools
Python, Scikit-learn, Jupyter Notebook
